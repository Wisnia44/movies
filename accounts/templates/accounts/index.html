Movies web app

Web app to browse movie database

Project is hosted on heroku at https://intense-woodland-74165.herokuapp.com


Funcinalities:

  1. Browsing results are viewable on a website with basic layout with pagination
  2. Possibility to save favourites into local database.

Local using instruction:

  1. Clone the repository with $git clone https://github.com/Wisnia44/movies.git`;
  2. Build docker images and run containers $docker-compose up --build;
  3. Server is up and running at http://0.0.0.0:8000/ !
